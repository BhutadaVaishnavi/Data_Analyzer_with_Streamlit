# Data Analyzer Dashboard

Data Analyzer Dashboard is a simple Streamlit-based web application designed for quick and interactive exploratory data analysis. Users can upload CSV or Excel files, explore dataset structure through shape, columns, and preview sections, and perform analytics using multiple visualization options. The dashboard supports numeric, categorical, and combined categorical-numerical analysis with charts such as line, bar, pie, and scatter plots, making it useful for beginners learning data analysis and dashboard development.

## Features
- Upload CSV/Excel files
- Dataset overview (shape, columns, preview)
- Interactive analytics dashboard
- Multiple chart types:
  - Line
  - Bar
  - Pie
  - Scatter

## Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib

## Run
```bash
pip install streamlit pandas matplotlib openpyxl
streamlit run app.py
```
